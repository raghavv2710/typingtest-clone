# **App Name**: Type Ace

## Core Features:

- Timed Typing Test: Initiate a timed typing test with selectable text options.
- Real-time Metrics: Display real-time typing speed (WPM) and accuracy during the test.
- Error Highlighting: Highlight typing errors in real time to provide immediate feedback.
- Performance Summary: Present a detailed summary of the typing performance at the end of the test, including WPM, accuracy, and error rate.

## Style Guidelines:

- Primary color: Professional blue (#3498DB), evoking trust and focus.
- Background color: Off-white (#F0F8FF), providing a clean and non-distracting workspace.
- Accent color: Light blue (#85C1E9) for interactive elements and highlights to subtly draw attention without causing distraction.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text
- Use clean, minimalist icons for settings and navigation.
- A clean and structured layout ensures readability and focus during the test.
- Subtle animations for start/stop and result display, enhancing user experience.
